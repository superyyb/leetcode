2.为什么要import deque
    q.pop(0) 是 O(k) 用的是 Python list 当队列：
    r, c = q.pop(0) pop(0) 每次都会把后面的元素整体左移，复杂度是 O(队列长度)。
    最坏情况下会让整体复杂度从 O(mn) 变成接近 O((mn)^2)（非常慢）。
    最优写法：用 collections.deque，popleft() 是 O(1)

3.BFS里面有edge case要continue的话,记得一定把list在最开始转为set()再判断
    （lc433: if in bank, lc752: if in deadends）
     List:nei in bank每次都是线性查找O(n), nei in bank_set 就是 O(1)
